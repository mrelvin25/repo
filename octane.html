Octane

extension

About

The tweak was created to simulate the virtual home button that appears on iPhone 7's when the home button breaks. It is useful to people with broken home buttons.

code

Features

Adds a Small Button to the button of the screen. Tap to simulate single home press, double tap to invoke the AppSwitcher. Configure size, opacity and activator toggles in settings.

build

Compatibility

Should be compatible with all iOS versions with support for homescreen widgets.
